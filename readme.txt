== Description ==

**Products Attachments For Woocommerce** using to setup attachments product in woocommerce product page to make it easy to access part of product.

> **Features**
>
> * Easy Setup
> * Change tab name in single product page
> * Attachment will be show by role wise
> * In order details Enable/Disable Option
> * My Account Page / Thank you Page with Enable/Disable Option
> * Attachemnt show after expire day option
> * Customize Name and Description of attachment
> * Attachment visibility Pages
> * Show/Hide for Logged user
> * Setup By order status
> * Setup By Expire Date of attachment
> * Download and View button text customized.
> * Download and View button text customized.
> * Color and text size Customized.
> * Multiple attechment add in one product.
> * pdf. png, jpg , all supported media in attechment.
> * manually prodduct to add attechment
> * Set Expire date/time for attechment



> *  **Products Attachments For Woocommerce** these files are automatically available to download in the product pages and/or on order details page after the customers have purchased them . You can also specify an Expiring time (only for order attachments)!
